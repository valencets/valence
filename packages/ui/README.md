# @valencets/ui

18 Web Components built on the ValElement protocol base class. ARIA roles, i18n, telemetry hooks, and hydration directives are built in. Ships with OKLCH design tokens, a Tailwind preset, and a theme contract for consistent styling across applications. Zero dependencies.

368 tests. [Full documentation on the wiki.](https://github.com/valencets/valence/wiki/Packages:-Ui)

## Quick Start

```bash
pnpm install
pnpm build
pnpm --filter=@valencets/ui test
```
