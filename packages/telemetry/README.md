# @valencets/telemetry

Summary queries, daily aggregation, and fleet data types for Valence telemetry. Builds on `@valencets/db` to turn raw event streams into queryable rollups.

96 tests. [Full documentation on the wiki.](https://github.com/valencets/valence/wiki/Packages:-Telemetry)

## Quick Start

```bash
pnpm install
pnpm build
pnpm --filter=@valencets/telemetry test
```
